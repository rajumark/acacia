Installation

Get started with Acacia in 3 simple steps.

1
Add Plugin
Add Acacia plugin to your app's build.gradle.kts file.
plugins { id("com.acacia") version "0.1.0" }
2
Import Functions
Import generated DSL functions in your Composables.
import com.acacia.generated.*
3
Start Using
Write cleaner, more concise Compose code.
Box( Modifier.fmw().fmh().bg(Color.Blue).p(16.dp) ) { Text("Hello Acacia!") }

About Acacia

Acacia is an AI-Native Compose DSL plugin that transforms verbose Jetpack Compose code into concise, readable DSL functions.

Features

• 391+ DSL functions for comprehensive coverage
• 70% less code with same functionality
• AI-friendly design for external model integration
• Cross-platform foundation for future expansion
• Performance optimized with intelligent caching
• Type-safe and reliable
