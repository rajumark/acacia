How it works

Three steps. One plugin. No config files.

01 — Install
Add one plugin line
Apply the ComposeWind Gradle plugin to your project. That is all the setup you need.
02 — Scan
KSP reads your classpath
KSP finds every Modifier extension in your project — from Compose itself, Material3, and your own custom modifiers.
03 — Generate
Wrappers appear automatically
ComposeWind generates compact Kotlin extension functions for every modifier found. Real code, full IDE support.
04 — Write less
AI generates in fewer tokens
Use the short API yourself or let AI generate it. Both paths compile to identical bytecode with zero runtime cost.

One line to install

Add the plugin, run a build. Everything else is automatic.

build.gradle.kts
plugins { id("io.composewind") version "1.0.0" }
Then just use it
// Before Modifier.fillMaxWidth().padding(16.dp).background(...) // After — ComposeWind Modifier.wFull().p(4).bgSurface()
